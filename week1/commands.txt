создаём репозиторий на  git hab  

переходим в папку с переходом 
 git init - превращаяет текущую папку директорию в git репозиторий
 git remote add название_связи (обычно - origin)
 ссылку _на_удаленный_репозиторий(SSH: git@github:...)
 git remote -v - просмотр списка удаленных репозиториев
 git remote rm - название _ связи (обычного - origin) -
 удаление связи с удаленным репозиторием
 git status -  проверка статуса файлов
 git add название_файла - отслеживание конкретных файлов

 git commit -  создание коммита (версии/слепка кода)
 git commit -m  'описание версии' - создание коммита с компютера 

 git push название_связи(обычно - origin) название_ветки - отправка версии кода


 git clone ссылка_на_удаленный_репозиторий - склонировать/скопировать/скачать удаленный репозиторий

git diff просмотрь внесенных изменений

git stash  - спрятать локалюные данный во временную область

git stash list -  просмотр списка
git stash apply - запись спрятанных данных/
извлечение спрятаггых данных
git stash drop - удаление спрятанных данных

git log - просмотр коммита/версий
git log --oneline -
